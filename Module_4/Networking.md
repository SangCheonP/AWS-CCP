# 모듈 4: 네트워킹

## Amazon VPC (Virtual Private Cloud)
- AWS 리소스를 격리된 네트워크 환경에서 실행

## 서브넷
- 퍼블릭 서브넷: 인터넷 연결 가능
- 프라이빗 서브넷: 내부 네트워크에서만 접근 가능

## 인터넷 게이트웨이 및 VPN
- **인터넷 게이트웨이**: VPC를 인터넷과 연결
- **가상 프라이빗 게이트웨이**: 프라이빗 네트워크와 연결

## 보안 그룹 vs 네트워크 ACL
- **보안 그룹**: 스테이트풀, 특정 EC2 인스턴스 보호
- **네트워크 ACL**: 스테이트리스, 서브넷 레벨에서 보호
