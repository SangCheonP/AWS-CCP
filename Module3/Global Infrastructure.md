# 모듈 3: 글로벌 인프라 및 신뢰성

## 리전 (시험 문제)
- AWS가 전 세계에서 데이터 센터를 클러스터링 하는 물리적 위치
- 각 리전은 여러 개의 가용 영역(AZ)으로 구성

## 가용 영역 (AZ)
- 논리적 데이터 센터 그룹
- 리전 내의 단일 데이터 센터 or 데이터 센터 그룹
- 서로 물리적으로 분리되어 있음

## Amazon CloudFront (시험 문제)
- 글로벌 콘텐츠 전송 네트워크 (CDN) 서비스
- 엣지 로케이션을 활용하여 콘텐츠 캐싱 및 전송 속도 향상

## AWS Management Console, CLI, SDK
1. **Management Console**: 웹 기반 UI
2. **AWS CLI**: 명령줄 인터페이스, 자동화 가능
3. **AWS SDK**: 프로그램에서 AWS와 상호작용 가능
